How record translation works

01

Select First Language:

Begin by choosing the language in which the original audio or voice recording is spoken. This is the language you want to translate from.

02

Select Second Language:

After selecting the first language, choose the target language into which you want the voice recording to be translated. This is the language you want the translated audio to be in.

03

Upload the Recording:

Once you've selected both the source and target languages, you need to upload the voice recording or audio file that you want to translate.

04

Click on Translate Button:

After uploading the recording and confirming the selected languages, there is typically a "Translate" button that you need to click. This initiates the translation process, and the tool will then process the voice recording, translating the content from the source language to the target language.

How live translation works

01

Select First Language:

Begin by choosing the language of the ongoing live conversation or speech. This is the language being spoken and the language you want to translate from. Most live translation tools have a feature where you can set the source language.

02

Select Second Language:

Similar to the previous steps, choose the target language into which you want the live speech to be translated. This is the language you want the translated content to be in. Use the tool's interface to select the desired target language.

03

Start Live Translation:

Activate the live translation feature on the tool or application. This may involve clicking on a "Start" or "Live Translate" button. Some platforms may require you to initiate a live recording or streaming function to begin the translation process in real-time.

04

Listen to Translated Output:

As the live conversation or speech progresses, listen to the translated output in the selected target language. The tool should provide a continuous translation of the ongoing speech, allowing you to follow the conversation in real-time.
